## 2. 快捷键

### 2.1 搜索

- `Ctrl + F`：底部搜索框查找
- `Ctrl + Shift + F`：文件夹内查找
- `Ctrl + P`：打开搜索框
    - 打开文件：输入文件名
    - 跳转：`:数字`
- `Ctrl + G`：跳转，自动带`:`
- `Ctrl + R`：搜索函数，自动带`@`
- `Ctrl + :`：查找关键词，自动带`#`
- `Ctrl + Shift + P`：打开命令框
- `Esc`：退出
